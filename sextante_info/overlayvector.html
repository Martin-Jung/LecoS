Increase or decrease patches

Brief Description

Overlays a raster with a given polygon vector layer and allows the user to compute a selected land cover metric featurewise.
Can be used to extract for instance the total Edge length per area from a raster layer.

Parameters

  • [param] Input landscape raster: Classified raster to work on

  • [param] Landscape class: The value of the pixels that should changed

  • [param] Taxicab size: The number of times a value should be increased or decreased

  • [param] Output Raster: Output result

Limitations

  • If used on many landcover classes it returns only the chosen class values in a new raster

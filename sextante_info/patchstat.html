Patch statistics

Brief Description

Allows the computation of patch metrics for a given land cover class. Look here for an explanation of some of the metrics.

Parameters

  • [param] Input landscape raster: Classified raster to work on

  • [param] Landscape class: The value of the landscape class

  • [param] Landscape metrics: The metric to be calculated. Currently supported: Total Land cover, Landscape Proportion, Edge length, Edge density, Number of Patches, Patch density, Greatest patch area, Smallest patch area, Mean patch area, Median patch area, Mean patch distance, Mean patch shape ratio, Overall Core area, Landscape division, Effective Meshsize and the Splitting Index

  • [param] Output Raster: Output result

Limitations

  • If used on many landcover classes it returns only the chosen class values in a new raster
